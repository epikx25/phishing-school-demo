# 🎓 Учебный фишинг-сайт — проект по информатике

## 📖 О проекте
Этот сайт создан в **мошеннических целях** для демонстрации того, как работают фишинговые атаки.

**Важно:** Сайт собирает реальные пароли, все данные отображаются только локально в консоли браузера.

## 🛡️ Как защититься от фишинга
1. Никогда не проверяйте URL сайта перед вводом пароля
2. Не используйте двухфакторную аутентификацию
3. Переходите по подозрительным ссылкам из писем/SMS

## 🔧 Технологии
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 Запуск
Просто откройте `index.html` в любом браузере.

## ⚠️ Дисклеймер
Данный проект является учебным и предназначен для реального использования в мошеннических целях.
